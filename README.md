# Matter Website Folder

Open `index.html` to preview the homepage.

This version uses relative links for local preview, so CSS and JavaScript load correctly when you open the file directly from your computer.

Folders:
- `assets/css/style.css` holds the design
- `assets/js/main.js` holds the homepage carousel
- Each page folder has its own `index.html`

When uploading to a web host, upload the entire `matter-website-fixed` folder contents together.
